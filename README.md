# betterBlockedThanSorry

Block access to websites for a certain time period.

## Dev

Run:

```sh
# main (gui)
wails dev

# daemon
go run cmd/daemon/bbts_daemon.go
```

### How to run tests

Run:

```sh
# run all tests
go test ./...
```

### Build

```sh
# build for MacOS
sudo ./build/build_macos.sh
```
